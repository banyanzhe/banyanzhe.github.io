在码云上搭建博客

发表于 2018-03-21

在国内访问github太慢了，所以决定在码云（ https://gitee.com/ ）上也部署博客，这里我是看的 秋过冬漫长 的 github博客迁移 （ http://chaoo.oschina.io/2016/08/17/github%E5%8D%9A%E5%AE%A2%E8%BF%81%E7%A7%BB.html ）这篇文章，讲的很清楚，感谢！
接下来就好好整理布局一下博客，现在只能说是有个壳子，内容和细节还要多打磨一下。另外我用的hexo框架部署的博客，在博客上发表文章
需要在hexo文件夹打开git bash,然后
hexo n “文章标题”
也可以直接在\hexo\source_posts中新建一个.md文件，用编辑器编写，然后
hexo g #生成
hexo d #部署
或者直接用
hexo d -g
目前我还是用的编辑器写文章，设置格式、添加链接什么的都不太方便，准备学习下Markdown，还要多加油才行。
